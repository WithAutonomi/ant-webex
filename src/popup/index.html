      Already installed antd? Find & run it

      If you set up antd before, it may just be stopped — or running without the option the extension needs. The extension reaches antd over http://localhost:8082 and needs it started with --cors.

      1 · Open a terminal

      2 · Start the daemon

      antd --cors

      Leave this window open while you browse. Seeing “command not found”? antd isn't on your PATH — run it with the full path shown below instead (still adding --cors).
